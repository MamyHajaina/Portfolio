Project information

  • Category: Application web
  • Établissement: Picolino
  • Project URL: https://preprod-modu3d.picolinoagency.com/
  • Technologies utilisées: Condeignter 4, WordPress, Python
Développement d'une plateforme de gestion et de rendu 3D de produits

Ce projet vise à fournir une solution complète pour la gestion de produits en 3D, adaptée aux besoins des entreprises souhaitant valoriser visuellement leurs articles. Il permet :

  • Gestion des produits 3D :
    • Ajout, modification et suppression de modèles 3D
    • Organisation par catégories, étiquettes et caractéristiques techniques
    • Visualisation en temps réel des modèles dans un viewer intégré
  • Rendu automatique :
    • Génération automatique de rendus visuels depuis les modèles 3D
    • Intégration de différents angles de vue et résolutions
    • Exportation vers plusieurs formats (images, vidéos, liens intégrables)
  • Fonctionnalités avancées :
    • Interface d'administration personnalisable via WordPress
    • Gestion sécurisée des utilisateurs et des rôles
    • Support des scripts Python pour traitement et analyse des modèles
